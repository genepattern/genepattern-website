GenePattern 3.9.6 is available for use on our Public GenePattern Server, via download, and as an Amazon Machine Image (AMI). The Release Notes give details on new features, fixes and modules released since September 25, 2015. These details include:

  • Added automatic R Package installer
  • The updated beta release of the JavaScript version CLSFileCreator
  • The beta release of modules to aid in the use of read count data in differential expression analysis: MergeHTSeqCounts and PreprocessReadCounts.

Feel free to contact us with any questions or comments.
