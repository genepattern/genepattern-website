GenePattern 3.8.0 has been released! 

3.8.0 (build 12839)  is available for use on our Public GenePattern Server,  via download and as an Amazon Machine Image (AMI)

New features include:

  • An updated User Interface
  • New and improved Modules & Pipelines Pane
  • Pinning favorite modules
  • Running a batch job over multiple dragged in files
  • Parallel execution of jobs in a pipeline
  • Pipeline designer improvements allowing the use of regular expressions to specify output files to be used as downstream input.
  • Multiple categories for individual modules and pipelines
  • Directories in dynamic drop downs
  • Setting of file purge intervals based on user or group

We also made some user requested fixes and enhancements:

  • A bug in GenePattern's SGE integration prevented job cancelation. This has been resolved.
  • Previously the pipeline designer did not allow non-admin users with createPublicPipeline permission to create public pipelines. This has been resolved.

A video overview covering some of these new features can be found here.

Full details about 3.8.0, including hardware/software requirements and known issues can be found in our release notes.
