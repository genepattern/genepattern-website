UPDATE MARCH 17, 2017 : Java applet visualizers no longer work in any browser. Please click here to read the blog post.

In a previous blog post from April we provided a work around for the the Chrome NPAPI plugin deprecation, which prevented our Java applet based visualizers from working.

As of September 1, 2015, this work around is no longer valid and Java applet based visualizers simply do not work in Chrome. 

https://support.google.com/chrome/answer/6213033?hl=en

To this end, the GenePattern team has been replacing our older Java applet based visualizers with new JavaScript based versions of the same name.

 

To date, the following have been updated:

  • CLSFileCreator
  • ComparativeMarkerSelectionViewer

 

Soon to be released:

  • HeatMapViewer
  • IGV

 

If you are experiencing any other issues with visualizers, not in Chrome, please review our FAQ.

 

Please contact us with any questions you may have, and follow us to get updates as we release these new JavaScript viewers.
