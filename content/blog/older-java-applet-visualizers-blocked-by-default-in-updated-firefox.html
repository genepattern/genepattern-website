UPDATE MARCH 17, 2017 : Java applet visualizers no longer work in any browser. Please click here to read the blog post.

Recent updates to Firefox may result in older GenePattern Java applet visualizer failing to launch.

A work around, posted on Java.com, can be found below. (additional work arounds are also available on this page)

(Note that while the webpage indicates Firefox 42 & Windows 8, this resolved my issue in Firefox 38 on Windows 7)

 

Option 1 (Simplest method)
Add property name dom.ipc.plugins.java.enabled in Firefox preferences
  1. Launch Firefox
  2. Type about:config in the address bar
  3. You will be presented with a Firefox warning about the configuration settings. After acknowledging the Firefox alert, you should see a listing of the configuration preferences.
  4. Right-click anywhere inside the displayed list of preferences. Select New then select Boolean.
  5. Add property name as dom.ipc.plugins.java.enabled 
    Add Preference name as dom.ipc.plugins.java.enabled.
  6. A window will be displayed, select false. Then select OK.

As always, please contact us with any questions or concerns you may have.

 

Lastly, please note that we are actively working to create JavaScript versions of all of our visualizers modules, so as to avoid these issues all together.
