GenePattern 3.9.7 is available for use on our Public GenePattern Server, via download, and as an Amazon Machine Image (AMI). The Release Notes give details on new features, fixes and modules released since January 22, 2016. These details include:

  • The beta release of the JavaScript version of HeatMapViewer
  • Updated releases of GSEA, GSEAPreranked and ssGSEAProjection, to provide access to MSigDB v5.1
  • Better retry handling in queueing system integration. Fixes "error getting status for job" failure.
  • GenomeSpace username and token can now be used to authenticate when using the REST API.
