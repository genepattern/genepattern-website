Normalizes RNAi hairpin scores across cell lines using ZMAD, PMAD, rank or quantile- ignores NaNs

Author: Barbara Weir, Pablo Tamayo, Aviad Tsherniak;The Broad Institute

Contact: bweir@broadinstitute.org

Algorithm Version:

Introduction

This module takes the data and puts the cell line data ranges on the same scale. For each cell line, either quantile normalize to the mean distribution of all cell lines, or subtract the median (ZMAD) or peak/mode (PMAD) and divide by the MAD. 

References

Cheung et al. PNAS 2011 Jul 26;108(30):12372-7. 

Parameters

Name Description
gct file* Input GCT file after shRNA scoring
method* Method of normalization. Allowed values: quantile, PMAD, ZMAD. Default value = quantile
plot filename* Output filename for plot showing before and after normalization

* - required

Input Files

  1. Input shRNA data file
    •  Input data file of cell lines (columns) and shRNAs (rows) in gct file format.

Output Files

  1. Output data file               [method]_[InputshRNA data filename].gct
    •  A normalized (quantile, PMAD, or ZMAD) data file of cell lines (columns) and shRNAs (rows) in a gct file format.
  2. plotfile          plot filename
    • A density plot showing the shRNA distribution before and after normalization for each cell line.
