Description

Displays four graphs to help visualize leading-edge subsets of selected gene sets in GSEA results.

Module Summary

For a full description of the GSEA algorithm, refer to the GSEA GenePattern module documentation or the GSEA User Guide.

As described in the Gene Set Enrichment Analysis (GSEA) paper (see References), the "leading edge" describes a subset of genes in a gene set. This subset contains the genes that appear in the ranked gene set list between zero and the point at which the running sum statistic reaches its greatest deviation from zero, as shown in the following figure.

The leading-edge subset can be interpreted as the core group of genes that accounts for the gene set’s enrichment signal.

This visualizer displays four graphs that help you visualize the overlap between the leading-edge subsets of selected gene sets.  You can also use the visualizer to generate an HTML report of the Leading Edge Analysis.

For more information on the leading edge analysis, see Interpreting leading edge analysis results in the GSEA User Guide.
