Reads rows of a CN file and removes outliers on a sample-by-sample basis using improvised method.

Author: Nicolas Stransky

Summary

This module removes outliers from copynumber data. If a point is an outlier (in both additive and multiplicative space) relative to the 5 points before and after it, it will be replaced by default with an NA. This mostly impacts probes that correspond to very low copynumber.

A point is considered an outlier if ALL of the following apply:

  • value != NA
  • abs (value - median_of_last_5_points) > 0.3
  • abs (value - median_of_next_5_points) > 0.3
  • abs (ln(value) - ln(median_of_last_5_points)) > 4
  • abs (ln(value) - ln(median_of_next_5_points)) > 4

If within 5 of the end of the chromosome, the 5 points the median is taken of is padded with copies of the value.

Parameters

Name Description
input file * A CN file (.cn, .RData, or .txt)
num probes * The number of trailing and leading probes to use to compute the median
outlier probe replacement * The replacement value for the outlier probe
multiplicative threshold * Multiplicative threshold
additive threshold * Additive threshold
output prefix * A prefix for the output file

* - required

Input Files

  1. <input.file>
    A copy number file in .cn, .RData, or .txt format.

Output Files

 

  1. <output prefix>.no_outliers.cn - A CN file with outliers removed
  2. <output prefix>.no_outliers.RData - An RData file with outliers removed for quick loading into R
