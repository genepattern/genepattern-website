Matches Fluidigm fingerprint report to reference set of CCLE SNP6.0 array data. Takes old Fluidigm software report or new PED style report (saved as .csv). Also backwards compatible with Sequenom reports (Combine 100k and AffySNP6.0 reports together into one file first)

Author: Barbara Weir, adapted from Nico Stransky;The Broad Institute

Parameters

Name Description
* format New PED style report OR Old Fluidigm software format OR Sequenom style report (expects combined 100k and SNP6.0 file) Default: new Fluidigm

* - required

Input Files

  1. platereport
    A fingerprinting plate report file saved as a .csv file.  This can be in 3 formats:
    1. new Fluidigm - A PED file from Fluidigm results, saved as a .csv file
    2. old Fluidigm - A file from the Fluidigm software, saved as a .csv file
    3. Sequenom - A file from the Sequenom software (100k and SNP6.0 panels combined), saved as a .csv file
  2. platemapping
    A fingerprint plate mapping file, necessary for the 'Sequenom' style report or the 'old Fluidigm' style report. Optional for the 'new Fluidigm' report.  An example file is provided - this file must have the columns:
    Well Sample.ID Sample.ID_2 Collaborator.Sample.ID Collaborator.Participant.ID Collaborator.Sample.ID_2

Output Files

  1. [AssayType]_Genotype.csv
    List of calls for each Cell line/SNP locus pair in both numeric and alphabetic format, along with orientation of the SNP.
  2. Master_Genotype_[AssayType]_panel.csv
    Matrix of calls for each Cell line/SNP locus pair in alphabetic format.
  3. SNP6to[AssayType]_all_results.csv
     List of cell line fingerprinted, # of SNPs used to match, identiity to and name of each cell line run with SNP6.0 array.
  4. SNP6to[AssayType]_matches.csv
     List of cell line fingerprinted, # of SNPs used to match, identiity to and name of each cell line run with SNP6.0 array that matches > 80%.
  5. SNP6to[AssayType]_topmatches_persample.csv
     List of cell line fingerprinted, # of SNPs used to match, identiity to and name of each cell line run with SNP6.0 array that matches the most.

=> 'AssayType' is determined by the input plate report file (either Sequnom or Fluidigm).

Example Data

Example data using the new Fludigm report can be found at the following link: ftp://gpftp.broadinstitute.org/example_data/modules/Achilles/FPmatchingExampleData_NewFluidigmParameter.zip

The input parameters are:

  • platereport -- Achilles_FPFluidigm_PlateReport_ped.csv
  • format -- 'new Fluidigm'

Example data using the Sequenom report can be found at the following link: ftp://gpftp.broadinstitute.org/example_data/modules/Achilles/FPmatchingExampleData_SequenomParameter.zip

The input parameters are:

  • platereport -- Achilles_FPSequenom_PlateReport_P1.csv
  • platemapping -- Achilles_FPSequenom_Platemapping_1.csv
  • format -- 'Sequenom'
