Converts an Agilent expression file (single channel only) into gct format.

Author: Marc-Danie Nazaire, Broad Institute

Contact: gp-help@broadinstitute.org

Algorithm Version:

Introduction

This module takes one or more single channel Agilent sample data and converts it into one gct file.

Parameters

Name Description
input file * One or more raw single channel Agilent files containing a ProbeName, Systematic Name, and either a gProcessedSignal or rProcessedSignal column
feature column * The name of the data column to use as the row identifier
description column * The name of the data column to use as the description column
signal column * The name of the data column containing the intensity values (i.e gProcessedSignal, rProcessedSignal)
output file * The prefix to use for the raw GCT output file name

* - required

Output Files

  1. <output.file>.gct
    A gct formatted file containing the raw Agilent intensity data.
